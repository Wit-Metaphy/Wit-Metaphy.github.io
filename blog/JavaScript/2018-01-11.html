# 正则表达式“不包含”怎么写

^((?!hello).)*$

(?!hello).匹配任意字符，要求字符前不出现hello，记为X ^(X)*$全部字符必须符合X的要求，若有一处不符合，那么说明原字符串中包含hello；若都符合，说明原字符串不包含hello

另一种写法：^(.(?!hello))*$

# Tip：

x(?!y) 正向否定查找 匹配'x'仅仅当'x'后面不跟着'y'。
