终端iterm2+ohmyzsh主题配置

July 09, 2018

下载安装iterm2和ohmyzsh

过程略

iterm2主题

preferences->Profiles->Colors中color presets选择Solarized Dark

ohmyzsh字体

fonts github链接

# clone
git clone https://github.com/powerline/fonts.git --depth=1
# install
cd fonts
./install.sh
# clean-up a bit
cd ..
rm -rf fonts

iterm2 preferences->Profiles->Text中勾选Use a different font for non-ASCII text

Font和non-ASCII Font字体选择14pt Meslo LG M Regular for Powerline

ohmyzsh主题

vi ./.zshrc

配置主题

ZSH_THEME="agnoster"

隐藏user@hostname

prompt_context(){}
